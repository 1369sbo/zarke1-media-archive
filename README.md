# zarke1-media-archive 

Generative/algorithmic video work; experimental; abstract.  

References the [Mash Limit](https://archive.org/details/@zarke1) page on the Internet Archive.  

Linking to the video uploads, which are described as "Experimental animation, abstract video art, and generative music-video projects by Mash Limit (aka Mushy/Zero01). Works created using open source software, code, command-line tools, software pipelines, and mathematical systems."

## Video production tools 

- [Python](https://www.python.org/) 
- [G'MIC](https://gmic.eu/#) 
- [FFmpeg](https://ffmpeg.org/) 
- [Krita](https://krita.org/en/) 
- [GIMP](https://www.gimp.org/)

&c..
  
Production scripts may be partial, reconstructed, renamed, or lost through years of iterative experimentation and migration between systems. There will be greater linkage of videos with actual processing steps and code going forward (late-May 2026).
